GENERAL PHOTO TIPS
  - Use good, even daylight (near a window, no harsh shadow), plain background.
  - Square photos (800x800) work best for product cards — crop before uploading.
  - Keep each file under 500KB so the site loads fast (see INSTRUCTIONS.md
    "Compressing photos" section for a free tool).
  - Save as .jpg (photos) — the site's code already expects .jpg; if you use
    .png just update the file extension in the matching HTML file too.
